elrpocket
=========

An Erlang library for Pocket API

## Fetch dependencies

Project depends on [git://github.com/davisp/jiffy.git) and [reloader](https://github.com/bjnortier/reloader).

    $ rebar get-deps

## Compile

    $ rebar compile

## Quick start

    $ ./start.sh
